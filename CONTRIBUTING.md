# Contributing to Boarda

Thank you for your interest in contributing to Boarda! We welcome contributions from everyone.

## Getting Started

1.  **Star the repository** on GitHub.
2.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/your-username/boarda.git
    cd boarda
    ```
3.  **Create a new branch** for your feature or fix:
    ```bash
    git checkout -b feature/your-feature-name
    ```

## Code Style

- We follow a strict MERN stack architecture.
- Please ensure your code follows the existing style (Space indentation, semi-colons, etc.).
- Run linting before committing (if available).

## Pull Request Process

1.  Ensure your code builds and runs locally.
2.  Update the documentation if necessary.
3.  Submit a Pull Request to the `main` branch.
4.  Provide a clear description of your changes.

## Code of Conduct

Please be respectful and kind to other contributors.
